# 📝 Cadastro Simples Responsivo

Este é um projeto de uma página de cadastro minimalista, desenvolvida com foco em experiência do usuário (UX) e validação de dados em tempo real. O formulário captura Nome, E-mail e Telefone, contando com máscaras dinâmicas para facilitar o preenchimento.

---

## 🚀 Funcionalidades

* **Validação de E-mail:** Utiliza o padrão nativo do HTML5 para garantir endereços válidos.
* **Design Responsivo:** A interface se adapta a diferentes tamanhos de tela (desktop, tablets e smartphones).
* **Feedback Visual:** Inputs com estados de foco e validação para guiar o usuário.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica dos campos.
* **CSS3:** Estilização moderna com Flexbox e transições suaves.
* **JavaScript (Vanilla):** Lógica de máscara de entrada e manipulação de eventos do formulário (sem dependências externas).
